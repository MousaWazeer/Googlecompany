Current IT Job Description:

1. Web Developer

  • Design, code, and modify websites from layout to function according to project specifications.
  • Collaborate with project managers and designers to create visually appealing sites that feature user-friendly design and clear navigation.
  • Test websites across browsers, operating systems, and devices for responsiveness and functionality.

2. Network Administrator

  1. Install, configure, and maintain network hardware and software systems.
  2. Monitor network performance and troubleshoot problems as they arise.
  3. Ensure network security by implementing and maintaining firewalls, VPNs, and other security measures.
  4. Provide technical support and training to users as needed.
